Global Causes of Death Analysis
This project analyzes global causes of death over time using Tableau to identify patterns, trends, and key insights.

Tools Used
- Tableau
- Excel

Project Overview
The goal of this project is to explore how different causes of death vary across regions and over time. The dashboard provides a visual breakdown of trends and highlights key patterns in the data.

Key Insights
- Identified leading causes of death globally
- Analyzed trends over time across different regions
- Used visualizations to highlight patterns and comparisons

Files
- Tableau Dashboard (.twbx)
- Dataset

How to View
Download the `.twbx` file and open it in Tableau to interact with the dashboard.
